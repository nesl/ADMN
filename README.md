# ADMN Code Repository

## Downloading the Files
Please find the relevant datasets (zipped pickle files) and MAE LayerDrop pretrained checkpoints [here](https://ucla.box.com/s/th57jrk1dh0x50nret2dyyvsg2ym6ywq).

## Environment Setup
Run `conda env create -f environment.yml`, and then `conda activate admn` to create the environment.

## Running the code
Please refer to the README files in [GTDM_Noise](GTDM_Noise/README.md), [GTDM_Lowlight](GTDM_Lowlight/README.md), [GTDM_Blur](GTDM_Blur/README.md), [MMFI_Noise](MMFI_Noise/README.md), [AVE_Custom](AVE_Custom/README.md).
